# Master Character Spec v0.1

## 目的

`reference/selected/` に配置された基準画像群から、観察可能な外見特徴のみを整理し、画像生成・PSD設計・監査レビューで参照するためのマスター仕様を定義する。

## 重要ルール

- 観察可能な特徴のみ記載する。
- 推測で補完しない。
- 年齢、BMI、体脂肪率、性格、職業、病歴、生活習慣は記載しない。
- 曖昧な箇所は「未確定」と明記する。
- 仕様更新時は、参照した画像ファイル名と観察内容を追記する。

## 監査レビュー観点

初回生成物 `outputs/master/front_v1.png` 以降は、以下の観点のみで仕様逸脱を確認する。

- 顔
- 体格
- 質感
- 衣装
- 小物

「似ているか」ではなく、「この仕様から逸脱していないか」を判定する。
